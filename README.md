# customer-behavior-analytics-pyspark
# Customer Behavior Analytics using PySpark

## Overview
This project analyzes customer behavior using PySpark to derive insights from customer spending data.

## Tools Used
- PySpark
- Databricks (conceptual)
- Python

## Architecture

1. Customer data is loaded from CSV
2. Data is cleaned and preprocessed
3. High-value customers are filtered
4. Aggregations performed to analyze spending patterns
5. Output used for business insights

## Features
- Data cleaning and preprocessing
- Filtering high-spending customers
- Aggregation using groupBy and avg
- Analysis of customer spending patterns

## Project Structure

- data/ : Contains customer dataset
- notebooks/ : PySpark analysis script

## Use Case
This project helps businesses understand customer behavior and spending trends for better decision-making.

## Note
This project is based on hands-on practice using PySpark and Databricks concepts.
